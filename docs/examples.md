# Examples

Runnable samples live in the repo **`examples/`** directory (not always shipped in the npm tarball — clone the repository to run them).

| File | Purpose |
|------|---------|
| [`examples/demo.ts`](../examples/demo.ts) | Broad tour: supervision, routers, FSM, event stream, mailboxes |
| [`examples/threaded-demo.ts`](../examples/threaded-demo.ts) | Worker pool + threaded behaviors |
| [`examples/examples.ts`](../examples/examples.ts) | Behavior factories for workers (`createCounterBehavior`, …) |

See [`examples/README.md`](../examples/README.md) for commands and notes on **absolute paths** for `behaviorModule`.
